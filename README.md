# Air-Health-Guardian
A full-stack Django app monitoring real-time air quality. It features automated sensor tracking, dynamic Chart.js data visualization, and a configurable threshold system to manage safety limits. Built with Python, MySQL, and CSS, it ensures a healthy environment through intelligent status alerts.
Overview
The Smart Indoor Air Health Guardian is a full-stack web application designed to monitor indoor air quality in real-time. The system tracks key environmental metrics such as temperature and humidity, ensuring a safe and healthy environment by providing immediate status alerts based on user-defined safety thresholds.

Key Features

Real-time Monitoring: Automatically fetches and displays sensor data with auto-refresh capability.

Dynamic Visualization: Interactive line charts powered by Chart.js to visualize temperature trends over time.

Configurable Thresholds: A user-friendly interface to update and save custom safety limits for temperature and humidity.

Status Alerting: Intelligent classification of air quality as 'Safe' or 'Alert' based on real-time readings.

Tech Stack

Backend: Python, Django Framework

Database: MySQL

Frontend: HTML5, CSS3, JavaScript (Chart.js)

Tools: Git/GitHub, VS Code
